# Test Files

This folder contains sample files for testing the PDF Converter application.

## Available Test Files:

1. **sample.txt** - Text file for testing text to PDF conversion

## How to Test:

1. Start the application
2. Register/Login
3. Upload any of these test files
4. Download the converted PDF
5. Verify the conversion quality

## Creating Your Own Test Files:

- **Images**: Any .jpg, .png, .gif, or .bmp file
- **Documents**: Word files (.docx, .doc)
- **Spreadsheets**: Excel files (.xlsx, .xls)
- **Presentations**: PowerPoint files (.pptx, .ppt)
- **Text**: Plain text files (.txt)

## File Size Limit:
Maximum file size is 20MB
